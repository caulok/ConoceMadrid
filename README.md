# ConoceMadrid

_Se trata de un proyecto que consiste en crear un simulador de reservas de alojamiento mediante el uso de HTML5, CSS3, frameworks como Bootstrap, pre-procesadores como SASS y JavaScript._

_En mi caso elegí como idea la web institucional de la ciudad de Madrid. La misma cuenta con: la página de inicio (index.html) y 2 páginas más: "Alojamiento" (aquella que contiene el simulador) y "Destinos"._

## Construido con 🛠️

* [Bootstrap v.5.0](https://getbootstrap.com/docs/5.0/getting-started/introduction/) - El framework de CSS.
* [SASS](https://sass-lang.com/documentation/) - Pre-procesador de CSS.
* [NodeJS](https://rometools.github.io/rome/) - Usado para procesar SASS.
* [OpenWeatherMap API](https://openweathermap.org/api) - API para el clima.
* [Font Awesome](https://fontawesome.com/) - Usado para colocar iconos.
